洗練されたデザインの会員登録・ログインフォームをショートコードで簡単に設置できるプラグイン。Google reCAPTCHA v3対応。

== Description ==

WordPress標準の `wp-login.php` のデザインではなく、サイトの世界観に馴染む「洗練された」会員機能を実装するためのプラグインです。

ショートコードを固定ページに貼り付けるだけで、ログインフォーム、会員登録フォーム、そして完全独自のパスワードリセットフローを導入できます。

Google reCAPTCHA v3 に対応しており、ユーザーにチェックボックスを押させることなく、ボットによるスパム登録や総当たり攻撃を防ぎます。

= 主な機能 =

* **ショートコード設置**: ログイン、登録、パスワードリセット申請、パスワード再設定の4つのフォームをショートコードで自由に配置。
* **洗練されたUI**: パスワードの表示切り替え（目のアイコン）や、リアルタイムの強度メーターを標準装備。
* **柔軟なリダイレクト設定**: ログイン後、ログアウト後、登録完了後のリダイレクト先を自由に設定可能。
* **登録フローのカスタマイズ**: メールアドレスのみでの登録（ユーザー名＝メールアドレス）や、登録直後の自動ログイン機能。
* **メールカスタマイズ**: 登録完了メールやパスワードリセットメールの件名・本文を管理画面から編集可能。
* **セキュリティ**: Google reCAPTCHA v3 を完全統合。
* **デザイン設定**: ブランドカラー（ベースカラー）を管理画面から設定し、ボタンやアクセントカラーに反映。

== Installation ==

1.  プラグインを `/wp-content/plugins/` ディレクトリにアップロードするか、プラグイン新規追加画面からインストールします。
2.  WordPressの「プラグイン」メニューからプラグインを有効化します。
3.  以下の4つの固定ページを作成し、それぞれのショートコードを本文に記述します。
    * ログインページ: `[edel_login]`
    * 会員登録ページ: `[edel_register]`
    * パスワードリセット申請ページ: `[edel_password_reset]`
    * パスワード再設定実行ページ: `[edel_password_new]`
4.  「設定 > Edel Membership」メニューへ進み、作成したページのURLやリダイレクト先、reCAPTCHAキーなどを設定してください。
5.  **重要**: 「【必須】パスワード再設定用ページのURL」には、`[edel_password_new]` を設置したページのURLを必ず入力してください。

== Frequently Asked Questions ==

= パスワードリセットメールのリンクをクリックしても「無効なアクセスです」と表示されます =
設定画面の「【必須】パスワード再設定用ページのURL」が正しく設定されているか確認してください。
また、URLの末尾にスラッシュ（/）が含まれているか、固定ページのパーマリンク設定と一致しているかを確認してください。WordPressのリダイレクト処理により、URLパラメータが削除されている可能性があります。

= reCAPTCHAのエラーが出ます =
Google reCAPTCHA Admin Console で「v3」のキーを発行しているか確認してください。v2のキーでは動作しません。また、サイトキーとシークレットキーが正しく入力されているか確認してください。

== Shortcodes ==

* `[edel_login]` : ログインフォームを表示します。ログイン済みの場合はログアウトボタンを表示します。
* `[edel_register]` : 新規会員登録フォームを表示します。
* `[edel_password_reset]` : パスワードを忘れたユーザーが、リセットメールを送信するためのフォームを表示します。
* `[edel_password_new]` : メール内のリンクからアクセスする、新しいパスワードを入力するためのフォームを表示します。
